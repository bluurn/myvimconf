# My vim config

This config includes:
 * my .vimrc
 * ag
 * ctrlp.vim
 * emmet-vim
 * vim-airline
 * vim-bundler
 * vim-colors-solarized
 * vim-fugitive
 * vim-rails

## Installation
    cd ~
    git clone https://github.com/bluurn/myvimconf.git
    cd myvimconf
    git submodule init
    git submodule update
    cp .vim ..
    cp .vimrc ..
